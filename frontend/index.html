Quick Start Guide

Steps to map and manage your layout boundaries

1

Upload Map Image

Drag and drop your layout plan image (.png, .jpg) into the left panel upload zone.

2

Extract Boundaries

Use **Local CV Mode** to fetch contours, or **AI Mode (Gemini)** to detect layout shapes and labels automatically.

3

Draw or Adjust

Select **Draw** tool to map points manually. Select **Adjust** to drag and edit vertex node handles directly.

4

Manage Plot Properties

Click any plot on the layout map to inspect dimensions, modify status, or log reservation contracts.

Need a standalone copy? Click **Export Standalone Map** at the bottom of the left panel to save an interactive HTML blueprint.

Workspace Guide

Map Navigation

Use the mouse wheel to **Zoom** the layout. Click and drag the map background to **Pan** around.

Canvas Tools

  • Select: Click any plot boundary on the map to inspect specifications or adjust reservation logs.
  • Draw: Click layout points sequentially to outline a custom polygon area. Click the first node to close.
  • Adjust: Adjust individual vertices handles by dragging nodes to calibrate shapes.
  • Delete: Select delete tool and click a plot polygon to delete it from layout coordinate tables.

AI & CV Extraction

Use **CV Extraction** for computer vision vector traces. Use **AI Mode (Gemini)** to automatically read labels, plot boundaries, and properties from layouts.
